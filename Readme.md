# 🤖 GitHub Copilot on GitHub.com

Everything Copilot can do — right in your browser, no IDE needed.

| Feature | How to Try It |
|---------|--------------|
| 💬 **Chat** | Click ✨ anywhere on GitHub and ask anything about your code |
| 🔍 **Code Review** | Open a PR → Reviewers → "Request a Copilot review" |
| 🛡️ **Autofix** | Security → Code scanning alerts → "Generate fix" |
| ✍️ **PR Summary** | New PR → click ✨ next to the description → "Generate summary" |
| 🏗️ **Workspace** | Open any issue → "Open in Copilot Workspace" |
| 🧩 **Extensions** | `@datadog`, `@sentry`, `@docker` and more in Copilot Chat |

## Try It Now

1. **[Request a Copilot review](../../pulls)** on this PR — see it find bugs in `examples/code_review_example.py`
2. **Ask Copilot Chat** → *"Explain the code in examples/refactor_example.ts"*
3. **Check the Security tab** → Copilot Autofix flags issues in `examples/autofix_example.js`

## Slash Commands

`/explain` · `/fix` · `/tests` · `/doc` · `/simplify`
